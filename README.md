# Інструкція для вчителя — Тест на 12 балів

Коротко: тест тепер знаходиться на окремій сторінці `quiz.html`. Це автономні HTML/CSS/JS файли — не потрібен інтернет. Тест містить 12 запитань, 1 бал за кожну правильну відповідь, максимум 12 балів.

Як відкрити:

1. Відкрийте `index.html` у браузері.
2. Натисніть кнопку "Почати тест" — відкриється `quiz.html`.
3. Учень відповідає на запитання та натискає "Перевірити".

Як оцінювати:

- Система автоматично підраховує бали: кожна правильна відповідь = 1 бал.
- Якщо виведено `12 / 12` — повний бал (12).
- Після перевірки правильні варіанти підсвічуються зеленим, неправильно вибрані — червоним.

Додатково:

- Кнопка "Показати відповіді" відкриває вікно з переліком правильних варіантів.
- Кнопка "Очистити" скидає відповіді та прибирає підсвічування.
- Для друку (PDF/принтер) кнопки ховаються автоматично.

Зміна запитань:

- Відредагуйте `quiz.html` для тексту питань/варіантів та `script.js` для правильних відповідей (масив `answers`).

Якщо хочете, я можу:

- Змінити тематику запитань.
- Додати таймер або підрахунок часу виконання.
- Зробити збереження результатів локально (localStorage) або вивантаження у CSV.
